Audio Input Element - User Guide

The Audio Input Element allows you to simulate how your circuit would respond to an audio signal. Here's how you can use it:

Adding an Audio Input:

  • Find the Audio Input element in the components menu under "Inputs and Sources". (In the "Draw" menu, or the right-click menu.)

Loading an Audio File:

  • Right-click on the Audio Input element to access its settings, and click "Edit".
  • Click on "Choose File" to select a file.

Supported Formats

The formats supported are whatever your browser supports. This typically includes:
  • WAV - Waveform Audio File Format, known for uncompressed audio data.
  • AIFF - Audio Interchange File Format, similar to WAV, uncompressed.
  • Ogg - Ogg Vorbis, an open, lossy audio compression format.
  • MP3 - MPEG Audio Layer III, widely used for its good quality-to-file size ratio.
  • MP4 - MPEG-4 Part 14 container format which can contain audio:
    • AAC - Advanced Audio Coding, often found within MP4 containers.
  • FLAC - Free Lossless Audio Codec, provides lossless compression.

Adjusting Settings:

Max Voltage:

  • This setting determines the peak voltage of the audio signal that will be input into your circuit. Adjust this if your circuit operates at different voltage levels. The audio data will be scaled by this value. So if V is the peak voltage, the output will range from -V to +V. It will be less than this if the audio data in the input file has a smaller range.

Start Position:

  • Set where in the audio file you want the simulation to start playing from. This is useful if you only need a portion of the audio or want to skip the silent parts at the beginning.

Use Cases:

  • Testing how speakers or audio equipment would react to different sounds.
  • Simulating audio signal processing circuits like filters, amplifiers, or mixers.
  • Educational purposes to demonstrate signal behavior in electronic circuits.
